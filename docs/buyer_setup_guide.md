# 【1단계】 미리보기 APK 설치 + 기기 ID 보내기

## 1-1. APK 파일 받기

APK(`.apk`)는 **Gmail·카카오톡에서 직접 보내기가 차단**되는 경우가 많습니다. 아래 방법 중 하나를 사용하세요.

### 방법 1 — Google Drive (권장)

1. 판매자가 보낸 **Drive 링크** 열기
2. `ktx-tget-preview.apk` **다운로드**
3. **내 파일** → **다운로드** 폴더

### 방법 2 — 카카오톡 ZIP 파일

카카오톡은 `.apk` 는 「지원하지 않는 형식」으로 막지만, **`.zip` 은 전송 가능**합니다.

1. 판매자가 보낸 `ktx-tget-preview-kakao.zip` 저장
2. **내 파일** → **다운로드** (또는 **KakaoTalk** 폴더) → zip 파일 탭
3. **압축 해제** (또는 **Extract**) → `ktx-tget-preview.apk` 확인
4. 압축 해제가 어려우면 zip을 PC로 옮겨 풀거나, 판매자에게 **Drive 링크**를 요청하세요.

> **판매자:** `./scripts/build_preview_apk.sh` 실행 시 `dist/ktx-tget-preview-kakao.zip` 이 함께 생성됩니다. 카카오톡에는 **zip** 만 보내세요.

---

## 1-2. Play Protect 차단 해제 (설치 실패 시 **먼저**)

「Google Play 프로텍트가 기기 보호를 위해 앱을 차단했습니다」가 나오면, **경고만 무시하는 것으로는 설치되지 않습니다.**  
아래를 **설치 시도 전에** 한 번 진행하세요.

**방법 A — Play Protect 검사 끄기 (갤럭시·픽셀 공통)**

1. **Google Play 스토어** 앱 실행
2. 오른쪽 위 **프로필** → **Play 프로텍트**
3. 오른쪽 위 **⚙ 설정**
4. **Play 프로텍트로 앱 검사** → **끔**
5. (있으면) **유해 앱 감지 개선** → **끔**

**방법 B — 삼성 갤럭시 Auto Blocker (One UI 6 이상)**

1. **설정** → **보안 및 개인정보 보호** → **Auto Blocker** (자동 차단)
2. **승인되지 않은 출처의 앱 설치 차단** → **끔** (또는 설치할 때만 잠시 끔)

설치가 끝난 뒤 Play Protect·Auto Blocker를 다시 켜도 됩니다.

---

## 1-3. 「알 수 없는 앱」 설치 허용 (처음 한 번만)

**삼성 갤럭시 기준**

1. 설정 → 애플리케이션(또는 앱) → 오른쪽 위 점 세 개 메뉴 → 특수 액세스(또는 특별한 접근) → **출처를 알 수 없는 앱 설치**
2. APK를 열 **앱** 선택 → **내 파일** (권장)
   - Google Drive에서 받음 → **내 파일**
   - 카카오톡에서 받음 → **내 파일** 또는 **카카오톡**
3. **이 출처 허용** 켜기

---

## 1-4. 미리보기 APK 설치

1. **내 파일** → **다운로드** → `ktx-tget-preview.apk` 탭
2. **설치** 탭
3. 경고가 나오면 **자세히** → **그래도 설치** (또는 **무시하고 설치**)
4. 여전히 차단되면 §1-2 Play Protect·Auto Blocker를 다시 확인한 뒤 **1~3번 반복**

> 이 앱은 접근성(화면 읽기)을 사용하는 **개발자 배포 앱**이라 Play Store에 없습니다. Play Protect 경고는 흔하며, 판매자가 보낸 파일이 맞다면 위 절차로 설치할 수 있습니다.

## 1-5. 기기 ID 복사해서 개발자에게 보내기

ktx-tget 실행 → 화면 상단 기기 ID 복사, 개발자에게 보내기(예: a1b2c3d4e5f67890)

# 【2단계】 등록 APK 설치 (실제 사용용)

개발자가 다시 보낸 파일 다운로드 및 설치

## 2-1. 등록 APK 설치

1단계와 같이 **Play Protect(§1-2)·알 수 없는 앱(§1-3)** 설정 후 **내 파일**에서 설치합니다.

「기존 앱을 업데이트하시겠습니까?」 또는 덮어쓰기 → **업데이트** / **설치**


# 【3단계】 꼭 켜야 하는 설정 (갤럭시)

등록 APK 설치 후 한 번만하면 됩니다.

## 3-1. 접근성 (필수 — 이걸 안 켜면 매크로가 동작하지 않음)

APK로 설치한 앱은 처음에 접근성 토글이 **꺼져 있고** 「**제한된 설정으로 제어됨**」으로 보일 수 있습니다. **정상적인 보안 동작이지만, 이대로 두면 매크로는 절대 동작하지 않습니다.**

### 3-1-A. 제한된 설정 허용 (먼저)

**삼성 갤럭시 기준**

1. **설정** → **애플리케이션** (또는 **앱**) → **ktx-tget**
2. 오른쪽 위 **⋮** (점 세 개) → **제한된 설정 허용**
3. 잠금 화면(PIN·지문) 확인이 나오면 통과

> 메뉴에 「제한된 설정 허용」이 없으면: **설정** → **앱** → **ktx-tget** → **앱 정보** 화면에서 ⋮ 다시 확인

### 3-1-B. 접근성 켜기

1. **설정** → **접근성** → **설치된 앱** (또는 **설치된 서비스**) → **ktx-tget**
2. **ktx-tget 사용** 스위치 **켜기**
3. 경고 창 → **허용** / **확인**

또는 ktx-tget 앱 → **시작하기** → **접근성 설정 열기** 로 같은 화면 이동

### 3-1-C. 확인

- 접근성 화면에서 ktx-tget이 **사용** 상태여야 합니다.
- 「제한된 설정으로 제어됨」 문구가 **사라져야** 합니다.
- 앱 홈에서 **매크로 실행** 스위치를 켤 수 있어야 합니다.

> **미리보기 APK**만 설치한 경우: 접근성을 켜도 매크로 스위치가 막혀 있습니다. **등록 APK**를 받아 덮어쓰기 설치한 뒤 다시 시도하세요.

## 3-2. 알림 (필수, Android 13+)

1. ktx-tget 실행 시 **알림 허용**
2. **설정** → **앱** → **ktx-tget** → **알림** → 모두 켜기

**밤에 알림을 꼭 받고 싶다면 (추가)**

- 설정 → 소리 및 진동 → 알람 볼륨을 충분히 올리기
- 설정 → 앱 → 특수 액세스 → 전체 화면 알림 → ktx-tget 허용 (Android 14 이상)

## 3-3. 배터리 (선택 — 밤새 매크로가 끊기면)

1. 설정 → 배터리 → 백그라운드 사용 제한
2. 절대 절전 안 함 (또는 최대 절전 예외) → ktx-tget 추가

# 【4단계】 사용 시작

1. 코레일톡 앱이 같은 휴대폰에 설치·로그인되어 있어야 합니다.
2. 코레일톡에서 열차 목록 화면을 연 뒤, 원하는 출발 시각이 목록 맨 위에 오도록 맞춰 둡니다.
3. ktx-tget 앱에서
   - 매크로 실행 켜기
   - 제외 열차·알림 등 옵션 설정
4. 코레일톡 열차 목록 화면으로 돌아가 두면, 앱이 자동으로 새로고침을 반복합니다.
5. 예매 화면에 도달하면 알림이 옵니다. 그때부터는 직접 결제하시면 됩니다.
