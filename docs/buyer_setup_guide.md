# ktx-tget 설치 안내 (구매자용)

## 전체 순서 (한눈에)

| 순서 | 할 일 | 시간 |
|------|--------|------|
| 1 | Google Drive 등에서 **APK** 설치 | 약 5~10분 |

# 【1단계】 APK 설치

## 1-1. APK 파일 받기

APK(`.apk`)는 **Gmail·카카오톡에서 직접 보내기가 차단**되는 경우가 많습니다. 아래 방법 중 하나를 사용하세요.

### 방법 1 — Google Drive (권장)

1. 판매자가 보낸 **Drive 링크** 열기
2. `ktx-tget-release.apk` **다운로드**
3. **내 파일** → **다운로드** 폴더

### 방법 2 — 카카오톡 ZIP 파일

카카오톡은 `.apk`는 막지만 **`.zip`은 전송 가능**합니다.

1. 판매자가 보낸 `ktx-tget-release-kakao.zip` 저장
2. **내 파일** → **다운로드** 폴더 → zip 파일 탭
3. **압축 해제** → `ktx-tget-release.apk` 확인

---

## 1-2. Play Protect 차단 해제 (설치 실패 시 **먼저**)

「Google Play 프로텍트가 기기 보호를 위해 앱을 차단했습니다」가 나오면, 경고를 무시하는 것만으로는 설치되지 않습니다.  
아래를 **설치 시도 전에** 진행하세요.

**방법 A — Play Protect 검사 끄기 (갤럭시·픽셀 공통)**

1. **Google Play 스토어** 앱 실행
2. 오른쪽 위 **프로필** → **Play 프로텍트**
3. 오른쪽 위 **⚙ 설정**
4. **Play 프로텍트로 앱 검사** → **끔**

**방법 B — 삼성 갤럭시 Auto Blocker (One UI 6 이상)**

1. **설정** → **보안 및 개인정보 보호** → **Auto Blocker**
2. **승인되지 않은 출처의 앱 설치 차단** → **끔**

설치가 끝난 뒤 다시 켜도 됩니다.

---

## 1-3. 「알 수 없는 앱」 설치 허용 (처음 한 번만)

**삼성 갤럭시 기준**

1. **설정** → **애플리케이션** → 오른쪽 위 점 세 개 메뉴 → **특수 액세스** → **출처를 알 수 없는 앱 설치**
2. APK를 열 앱 선택 → **내 파일** 또는 **카카오톡**
3. **이 출처 허용** 켜기

---

## 1-4. APK 설치

1. **내 파일** → **다운로드** → `ktx-tget-release.apk` 탭
2. **설치** 탭
3. 경고가 나오면 **자세히** → **그래도 설치**
4. 여전히 차단되면 §1-2 Play Protect·Auto Blocker를 다시 확인

> 이 앱은 접근성(화면 읽기)을 사용하는 개발자 배포 앱이라 Play Store에 없습니다. Play Protect 경고는 흔하며, 판매자가 보낸 파일이 맞다면 위 절차로 설치할 수 있습니다.

---

# 【2단계】 기기 ID 복사 → 판매자에게 보내기

1. **ktx-tget** 앱 실행
2. 화면 상단 **「라이선스 활성화 필요」** 카드 확인
3. **내 기기 ID** 아래 값 확인 (영문·숫자 16자 내외)
4. **기기 ID 복사** 버튼 탭
5. 카카오톡 등으로 **판매자에게 전송**

> ⚠️ **설정의 일련번호·IMEI가 아닙니다.** 반드시 **앱에 표시된 기기 ID**를 보내주세요.

---

# 【3단계】 라이선스 키 입력 → 활성화

판매자가 보낸 라이선스 키(예: `A3F2-9C1B-E047-1D8F`)를 받으면:

1. ktx-tget 앱 실행
2. 상단 카드의 **라이선스 키** 입력란에 키 붙여넣기
3. **활성화** 버튼 탭
4. 「라이선스가 활성화되었습니다」 메시지 확인

활성화 성공 시 카드가 사라지고 매크로 스위치를 켤 수 있습니다.

# 【4단계】 꼭 켜야 하는 설정 (갤럭시)

## 4-1. 접근성 (필수 — 이걸 안 켜면 매크로가 동작하지 않음)

APK로 설치한 앱은 처음에 접근성 토글이 **꺼져 있고** 「**제한된 설정으로 제어됨**」으로 보일 수 있습니다. 이대로 두면 매크로가 동작하지 않습니다.

### ① 제한된 설정 허용 (먼저)

1. **설정** → **애플리케이션** → **ktx-tget**
2. 오른쪽 위 **⋮** (점 세 개) → **제한된 설정 허용**
3. 잠금 화면(PIN·지문) 확인이 나오면 통과

> 메뉴에 「제한된 설정 허용」이 없으면: **설정** → **앱** → **ktx-tget** → **앱 정보** 화면에서 ⋮ 다시 확인

### ② 접근성 켜기

1. **설정** → **접근성** → **설치된 앱** → **ktx-tget**
2. **ktx-tget 사용** 스위치 **켜기**
3. 경고 창 → **허용** / **확인**

또는 ktx-tget 앱 → **시작하기** → **접근성 설정 열기**

### ③ 확인

- 접근성 화면에서 ktx-tget이 **사용** 상태여야 합니다.
- 「제한된 설정으로 제어됨」 문구가 **사라져야** 합니다.

## 4-2. 알림 (필수, Android 13+)

1. 앱 실행 시 **알림 허용**
2. **설정** → **앱** → **ktx-tget** → **알림** → 모두 켜기

**밤에 알림을 꼭 받고 싶다면**

- **설정** → **소리 및 진동** → **알람** 볼륨 올리기
- **설정** → **앱** → **특수 액세스** → **전체 화면 알림** → ktx-tget 허용 (Android 14 이상)

## 4-3. 배터리 (권장 — 밤새 매크로가 끊기면)

1. **설정** → **배터리** → **백그라운드 사용 제한**
2. ktx-tget → **절대 절전 안 함**

# 【5단계】 사용 시작

1. 코레일톡 앱이 같은 휴대폰에 설치·로그인되어 있어야 합니다.
2. 코레일톡에서 열차 목록 화면을 연 뒤, 원하는 출발 시각이 목록 맨 위에 오도록 맞춰 둡니다.
3. ktx-tget 앱에서
   - 매크로 실행 켜기
   - 제외 열차·알림 등 옵션 설정
4. 코레일톡 열차 목록 화면으로 돌아가 두면, 앱이 자동으로 새로고침을 반복합니다.
5. 예매 화면에 도달하면 알림이 옵니다. 그때부터는 직접 결제하시면 됩니다.
